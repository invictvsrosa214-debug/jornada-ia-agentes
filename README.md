# Jornada IA e Agentes

Este repositório documenta minha jornada para me tornar especialista em Inteligência Artificial aplicada, DevOps com Azure, GitHub, Visual Studio Code e construção de agentes com GPT.

## Objetivos

- Aprender fundamentos de IA generativa
- Construir aplicações com GPT
- Criar agentes de IA
- Dominar GitHub e GitHub Actions
- Aprender Azure DevOps
- Criar pipelines de CI/CD
- Publicar projetos reais no portfólio

## Estrutura do repositório

- `docs/` — plano de estudos e documentos gerais
- `fundamentos/` — anotações sobre conceitos técnicos
- `projetos/` — projetos práticos da jornada
- `devops/` — estudos e práticas de DevOps
- `agentes/` — ideias e experimentos com agentes de IA

## Trilha principal

1. Fundamentos técnicos
2. Git, GitHub e VS Code
3. Python para IA
4. APIs com GPT
5. RAG e embeddings
6. Agentes de IA
7. GitHub Actions
8. Azure DevOps
9. Deploy em Azure
10. Projeto final de portfólio
